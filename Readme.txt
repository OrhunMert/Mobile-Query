**********PROJE ÖZETİ**********

->Bu proje bizden 3 farklı tipteki sorgulardan 1 tane seçip, bu sorguları mobil programlama dili ile çözmeizi ister, sorgular için gerekli
olan veriler ise bir cloud'dadır, 
=> Bizim seçtiğimiz sorguları söylecek olursak  
 -) 1. sorgu için “En uzun mesafeli 5 yolculuktaki gün ve mesafeleri listeleyiniz”
 -) 2. sorgu için “Günlük seyahat başına düşen ortalama alınan ücretlere göre; 
	en az ücret alınan iki tarih arasındaki günlük alınan ortalama ücretleri listeleyiniz”, 
 -) 3. sorgu için “En az 3 yolcunun bulunduğu seyahatlerden en kısa mesafeli ve en uzun mesafeli yolu çiziniz. Başlangıç ve varış konumları 
		lokasyonun merkezi kabul edip mesafeye göre bir yol bulunmalıdır”

**********PROJEYİ ÇALIŞTIRMAK VE TEST ETMEK**********

-> kaynakKod.zip içeriğini çıkarınız ve önerilen olarak Visual Studio code içerisinde açınız sorunsuz açılacaktır.
-> terminali açınız
-> ilk olarak npm i ile node modülü indirin veya updateleyin
-> vs code terminali kullanıyorsanız ilk önce "npm run start" ile metro server'ını çalıştırın, daha sonra npm run android ile android
	emülator veya telefonunuzda run edebilirsiniz 
-> cmd ile çalıştırmak istiyorsanız proje dosyasına gelin, ilk önce npm start ile metro server'ını çalıştırın, başka bir cmd ile proje
	dosyasına gelin ve react-native run-android kodu ile emülator veya telefonunuzda run edebilirsiniz
